MIPS commands. This is a list of commands for most of the MIPS modules.

General Commands
  NOP			Do nothing
  GVER			Returns the current version string
  ISPWR			Reports system on or off status
  SUPPLIES		Reports system power supply voltages if equiped with monitors
  V12			Reports 12 volt power supply volgate if equiped with monitors
  V24			Reports 24 volt power supply volgate if equiped with monitors
  CUR			Reports system current in amps if equiped with monitors
  PWR			Reports system power in watts if equiped with monitors
  UPTIME		Reports the number of minutes the system has been running
  GERR			Returns integer error code
  GNAME			Returns user assigned name string
  SNAME,name		Sets the system name to user entered string
  SSERWD,sec		Sets the serial watch dog timer in seconds, set to 0 to disable
  GSERWD		Returns the serial watch dog timer value in seconds
  UUID			Reports the microcontrollers unique ID, 128 bits, hex format
  ABOUT			Returns details about system modules installed
  SMREV,b,m,r	     	Defines the rev level of a module,  b=board, m=module address, r=rev level desired
  RESET			Reboots the MIPS system
  SAVE			Save general parameters to SD memory card
  GCHAN,module		Returns the number of channels for the selected module, RF,DCB,ESI,TWAVE,FAIMS,FIL
  MUTE,ON|OFF		Mute on will turn off all messages to host, messages from host will still be processed
  ECHO,TRUE|FALSE	Echo on will return the command before the response
  TRIGOUT,value     	This command controls the trigger output, values can be HIGH,LOW,or PULSE. Pulse will generate a 1 mS pulse.
                        If value is a numeric it will represent a pulse width in uS
  AUXOUT,value     	This command controls the auxiliary trigger output, values can be HIGH,LOW,or PULSE. Pulse will generate a 1 mS pulse.
  DELAY,value       	Delays the number of milliseconds provided
  GCMDS			List all commands
  GAENA			Returns analog input status, TRUE or FALSE
  SAENA,value       	Defines analog input status, TRUE or FALSE
  THREADS           	Lists all threads, debug command
  STHRDENA,nm,val 	Thread control, nm=name, val = TRUE or FALSE
  STHRDINT,nm,val	Sets thread interval in milliseconds,  nm=name, val = milliseconds
  SDEVADD,b,ad      	ADC address setup, b = board 0 or 1, ad = address, base 10
  RDEV,ch		ADC read, ch = channel 0 - 7
  RDEV2,ch		AD7994 ADC read, ch = channel 0 - 3
  TBLTSKENA,val		Enable IO in table mode, val = TRUE or FALSE
  LEDOVRD,val       	Override button LED, val = TRUE or FALSE
  LED,val		LED color, val = bit mask. 0=Red,1=Blue,2=green,3=blink
  DSPOFF,val        	Turns off display, val = TRUE or FALSE
  STRTDLY,val		Sets the startup delay in seconds
  CHKIMAGE,file	       	Checks status of bmp file on SD card
  LOADIMAGE,file	Displays bmp file that is saved on SD card
  BIMAGE,file       	Set an image file to be displayed on power up
  STATUS		Returns the reason for the last reboot and the trace buffer
  SSERIALNAV,value	Enabled and disables the remote host navigation, TRUE or FALSE
  TRACE			Issuing this command enables the trace buffer
  DIR			Displays a list of files on the root of the SD card
  DEL,filename		Deletes the selected file from the SD card
  GET,filename		Reads a file from SD card to host, issued only by MIPS app
  PUT,filename size	Send a file from the host to SD card, issused only by MIPS app
  SAVEMOD,file,b,a	Save EEPROM to SD file. file = file name, b = board address, a =address in hex
  LOADMOD,file,b,a	Load EEPROM from SD file. file = file name, b = board address, a =address in hex
  SAVEALL		Saves all module EEPROM data to individual files on SD card
  LOADALL		Loads all module EEPROM data from individual files on SD card
  GETEEPROM,brd,add	Reads EEPROM module contents to host file, issued only by MIPS
  PUTEEPROM,brd,add	Writes EEPROM module contents from host file, issued only by MIPS
  SSPND,TRUE|FALSE	Suspendes all background tasks, will limit functionality
  GSPND			Returns the suspend state
  CPUTEMP		Returns CPU temp in degrees C
  TWITALK,brd,add	Redirect the serial communications through a board's TWI (WIRE) interface,
                        brd and TWI add are base 10
  TWI1TALK,brd,add	Redirect the serial communications through a board's TWI (WIRE1) interface,
                        brd and TWI add are base 10. Note: brd address is required but not used
  SSER1ENA,val		Set the Serial1 port enable, TRUE = general use, FALSE for ethernet use or wifi
  GSER1ENA		Get the Serial1 port enable status
  SETADDRESS,add	Set memory address for memory access, add is hex, base 16
                        Note: this command will set offset to 0
  SETOFFSET,add		Set memory address offset for memory access, add is hex, base 16
                        This offset is added to the address before memory access
  SETEEPADD,nm,brd	This command will set the address to the configration structure for a selected module
                        nm = module name, brd = module board address
			Valid modules: FR,RFamp,DCB,ESI,TWAVE,FAIMS,FAIMSfb,HV,FIL,ARB,DAC
  WRITE,type,val	Write to memory, type = BYTE,WORD,DWORD,INT,FLOAT
  READ,type		Read from memory, type = BYTE,WORD,DWORD,INT,FLOAT
  DUMP			Displays the memory contents at address + offset in hex, displays 512 values
  FORMAT,brd,add,typ	Format the EEPROM, args example,A 0x50,RFdrvA R1
  REEPROM,b,twi,add	Reads a byte from the selected EEPROM. b=board select 0 or 1, twi=twi address base 10, add=EEPROM address base 10.
  WEEPROM,b,twi,add,d	Writes a byte to the selected EEPROM. b=board select 0 or 1, twi=twi address base 10, add=EEPROM address base 10, d=data base 10.
  SAVEM			Save the defined module to EEPROM
  TWIRESET		Resets the TWI interface
  TWIERROR		Reports the numner of detected TWI failures
  STWIHDW,val		If true then the TWI hardware interface is used for ADC read functions
  GTWIHDW		Returns the current status.
  STBLRETRIG,val	Set the table retriggerable flag, TRUE=retriggerable.
  GTBLRETRIG		Returns the table retriggerable flag, TRUE=retriggerable.
  SSETECHO,val		Set the SerialUSB to Serial1 echo, TRUE enables echo.
  GSERECHO		Return the SerialUSB to Serial1 echo, TRUE enables echo.
  POWER			Cycles power on the MIPS system, requires MIPS feature.
  SUBRDSEL,TRUE|FALSE	Sets the use of board select. If TRUE the MIPS system used board select, default is TRUE;
  GUBRDSEL		Return board select usage status.
  DREAD,pin		Reads the selected input pin, returns HIGH or LOW.
  DWRITE,pin,HIGH|LOW	Writes the selected output pin.
  DSET,pin,dir		Sets the mode of selected pin, dir=INPUT|PULLUP|OUTPUT
  DEBUG,val		Calls debug function and passes val (integer), 
  TWICMD,b,add,str	Sends a TWI string to wire interface, b=board address 0 or 1,add=TWI address(base 10),str=command string.
  TWI1CMD,add,str	Sends a TWI string to wire1 interface, ,add=TWI address(base 10),str=command string.
  TWISCAN,brd		Scans for detected devices on wire interface, brd=board address 0 or 1.
  TWI1SCAN		Scans for detected devices on wire1 interface.
  TWIADDSET,md		This function will automatically set the TWI addresses for the selected module. md=RFD|DCB|FAIMS
  STRPLVL,val		Sets the DC bias module trip level, val=percent of full scale trip level.
  GTRPLVL		Returns the DC bias module trip level.

The MIPS system can display and report and event log
  SLOGENA,TRUE|FALSE	This command enable logging if TRUE, default ir TRUE
  GLOGENA		Return the logging enable status
  LOGREP		Reports the logs oldest log entry, returns nothing if all log entries have been read

Real time clock functions. Note, no battery backup so
the clock resets every time the hardware boots
  STIME			Sets the current time, 00:00:00 format
  GTIME			Returns the current time, 00:00:00 format
  SDATE			Sets the current date, dd/mm/yyyy format
  GDATE			Returns the current date, dd/mm/yyyy format

Level detection module change enable
  ENALDET		Enables the level detection module to report a detected change to the host.
                        This function requires the module address in hex.

Pulse counter functions
  COUNT,inp,level	Setup pulse counter, inp=(Q-X or NA) and level= POS, NEG, BOTH, NA
                        NA disables
  GCNT			Returns the current pulse count
  CLRCNT		Clear the current count
  SCNTTRG,cnt		Set the counter threshold count, cnt = threshold
  GCNTTRG		Return the counter threshold count
  TRIGCNT,TRUE|FALSE	If TRUE generate 5uS trigger on Trig out at threshold count
  TRIGRST,TRUE|FALSE	If TRUE reset counter out at threshold count

ADC functions
  ADC,ch		Read DUE ADC channel, ch = channel 0 - 3
  ADCAVG		Report the average ADC value read by the change detection function. 
  			This uses the SADCSAMPS value to set the number of samples in the average
  ADCGAIN,ch,gain	Set ADC channel gain. Channel range 0 to 3, gains are 1,2, or 4
  ADCCHG,ch,thres	Enable ADC change detection. Channel range 0 to 3, threshold 0 to 100
  ADCCPRM,wcnt,rcnt	Change detection filter,wcnt=required number of event outside window,rcnt=required number within new window
  RADCCHG,gain		This function will report ADC value changes to the host, the ADC value is multiplied by gain
  ADCINIT		ADC vector recording setup
  ADCTRIG		ADC vector recording software trigger
  ADCABORT		ADC system abort
  SADCCHAN,ch		Set ADC channel number
  GADCCHAN		Get ADC channel number
  SADCSAMPS,num		Set ADC number of samples
  GADCSAMPS		Get ADC number of samples
  SADCVECTS,num		Set ADC number of vectors
  GADCVECTS		Get ADC number of vectors
  SADCRATE,num		Set ADC sample rate in Hz
  GADCRATE		Get ADC sample rate in Hz
